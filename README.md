# PL Briefing Archive

Weekly Premier League briefing emails, generated by a Claude Code scheduled remote agent.

## Schedule

- **When:** Every Thursday at 6:30 AM ET
- **What:** HTML email covering the latest PL matchweek — results, standings, fixtures, FPL picks, and analysis
- **Delivered to:** Gmail

## Archive Structure

Each file in `archive/` is a self-contained HTML email that can be opened in any browser.

Naming: `PL-Brief-MW[X]-[YYYY-MM-DD].html`
